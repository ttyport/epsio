Epsio
Идея:  Реализовать сайт знакомств для животных в практических целях, ведь на данный момент времени подобных ресурсов мне не встречалось.

Требуется реализовать:
Веб-интерфейс
Базу данных животных
Систему подбора наиболее подходящей пары
Создание профилей хозяев
Работоспособность на любом типе устройств (desktop, mobile)
Доступ из любой точки мира

Реализация:

Приложение реализовано на фреймворке flask

Для реализации было выполнено:
Установка и настройка сервера под управлением операционной системы ubuntu server
Установка и настройка веб-сервера apache2, интеграция c flask
Установка и настройка СУБД MySQL
Приобретение домена и настройка dns
Фронтенд (html, css, js, ajax)
Бекэнд (python-flask)
Индексация в Google

Основные страницы:
Login - форма логина
Register - форма регистрации
Home - страница знакомств
Profile - страница добавленных животных
Profile/add - страница добавление новых животных
Chat - страница чата 

Интересные фичи:
Реализация механизма сессий
Хэширование паролей с помощью sha256
Механизм подбора пары
Стилизация под известный сайт знакомств “псиндер”
При взаимной симпатии становятся доступными контактные данные
Проект защищен открытой лицензией GPLv3
